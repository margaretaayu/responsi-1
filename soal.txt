Tunjukkan dengan menggunakan Command Promt (CMD) di Windows, bagaimana cara berikut:
a) Memindahkan File
b) Menyalin File dan Folder
c) Menghapus File dan Folder

Buatlah sebuah repository di Github dengan nama responsi-1.
Di dalam repository responsi1, buatlah 2 branch baru dengan nama yang berbeda.
Kemudian, buatlah file untuk masing-masing branch tersebut, edit file tersebut setidaknya 2 kali commit.
Lakukan merge branch baru yang telah di buat ke dalam branch main (utama).

Lakukan clone repository github responsi-1 ke komputer lokal.
Lakukan inisialisasi git pada folder hasil clone tersebut dengan menggunakan git bash.
Buatlah setidaknya 2 file baru pada git di komputer lokal.
Lakukan Commit pada file tersebut.
Kemudian lakukan push folder git lokal ke repository responsi-1 di Github.



Jawab:
a)Untuk memindahkan file menggunakan perintah "move namafile folder_tujuan"
b)Untuk menyalin file menggunakan perintah "copy namafile folder_tujuan"
  Untuk menyalin folder yang di dalamnya terdapat file dan subfolder menggunakan perintah "xcopy /s namafile folder_tujuan"
c)Untuk menghapus file menggunakan perintah "del nama_file"
  Untuk menghapus folder menggunakan perintah "rmdir nama_folder"

1. Buka https://github.com/. Jika belum memiliki akun, silakan buat akun terlebih dahulu.
   Jika sudah selesai membuat akun atau sudah memiliki akun, ikutilah langkah-langkah sebagai berikut:
   klik pojok kanan atas (gambar profil) > pilih Your repositories.
2. Setelah masuk ke repository, pilih New > isi nama repository yaitu responsi-1
3. Kita akan membuat branch baru dengan langkah berikut:
   pilih Edit this file > tambahkan isi pada file > isi keterangan apa yang kita lakukan pada repository >
   pilih Create a new branch for this commit and start a pull request > isilah nama file >
   setelah berhasil isilah keterangan > klik Create pull request > klik Marge pull request
4. Edit file dengan langkah berikut:
   klik file seperti soal.txt > pilih Edit this file > ketik isi > commit file dengan keterangan langkah yang sudah dilakukan
   Kemudian jika ingin edit file lagi lakukan langkah-langkah seperti di atas
5. Kita akan marge branch baru dengan langkah:
   klik Contribute pada branch baru > kemudian akan otomatis masuk pada Pull request > klik Marge pull request > klik Confirm marge
   Setelah selesai marge warna akan berubah menjadi ungu. Jangan lupa cek pada branch main apakah sudah berhasil!
