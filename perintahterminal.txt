Perintah Dasar Terminal
Linux, MacOS, Unix, dll.

Navigasi
>cd nama_folder
 masuk ke dalam folder yang dimaksud
>cd ..
 kembali ke direktori / folder sebelumnya

Melihat Isi Direktori / Folder
>ls
 menampilkan semua isi yang ada di dalam sebuah direktori / folder

Membuat Direktori / Folder
>mkdir nama_folder
 membuat folder baru

Menghapus Folder dan File
>rm-r nama_folder
 menghapus direktori / folder
>rm nama_file
 menghapus file

Menyalin File
>cp namafile folder_tujuan
 menyalin file ke dalam folder yang dituju

Memindahkan File
>mv namafile folder_tujuan
 memindahkan file ke dalam folder yang dituju
